# AI Job Market Intelligence 2025 Dashboard

An interactive data analytics dashboard that presents insights into the global AI job market in 2025. The project focuses on salary trends, job demand, required skills, experience levels, work modes, and geographic talent distribution.

This dashboard is intended for job seekers, data analysts, recruiters, and researchers who want a data-driven understanding of the evolving AI employment landscape.

## Project Objectives

* Analyze global AI job salary trends
* Identify high-demand AI roles and skills
* Compare salaries across experience levels and company sizes
* Visualize remote, hybrid, and on-site work distribution
* Highlight global AI talent and salary hotspots

---

## Key Insights

### Salary Analysis

* Average and maximum salary benchmarking
* Salary comparison by company location
* Salary trends by technical skills
* Salary distribution across company sizes

### Global Talent Hotspots

* Geographic visualization of AI talent demand
* Identification of high-paying regions

### Skills and Role Demand

* Salary curve by technical skills
* Most in-demand AI roles:

  * Machine Learning Engineer
  * Deep Learning Engineer
  * Computer Vision Engineer
  * Data Engineer
  * Data Scientist
  * AI Product Manager

### Work Mode Distribution

* Remote vs Hybrid vs On-site job distribution
* Employment type analysis:

  * Full-time
  * Contract (fixed-term)
  * Freelance

### Career Growth

* Salary vs experience analysis
* Comparison across entry-level, mid-level, senior, and executive roles

## Interactive Filters

The dashboard supports dynamic filtering based on:

* Industry
* Company location
* Employment type
* Experience level
* Work mode

All visualizations update automatically according to selected filters.

## Dataset Description

The dataset includes the following attributes:

* Job title
* Salary (USD normalized)
* Company size
* Industry
* Company location
* Experience level
* Required technical skills
* Work mode (Remote, Hybrid, On-site)

The dataset may be curated from public sources or synthetically generated for analysis purposes.
